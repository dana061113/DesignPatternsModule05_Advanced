# DesignPatternsModule05_Advanced

Курс: Шаблоны проектирования приложений  
Тақырып: Модуль 05 – Порождаушы паттерндер (Singleton, Builder, Prototype)

---

## Жобаның мақсаты

Бұл жобада үш негізгі порождаушы паттерннің күрделі нұсқалары жүзеге асырылды:

1. Singleton – көпағынды ортада жұмыс істейтін Logger
2. Builder – стильдері және бөлімдері бар көпформатты есеп жүйесі
3. Prototype – онлайн ойын персонаждарын терең көшіру

---

# 1️⃣ Singleton – Logger

Logger класы:
- Бір ғана экземпляр құрады
- Multithreading кезінде қауіпсіз
- Файлға лог жазады
- LogLevel (INFO, WARNING, ERROR) қолдайды
- Конфигурацияны сыртқы файлдан жүктей алады
- LogReader арқылы логтарды фильтрациялап оқуға болады

---

# 2️⃣ Builder – Report System

Builder паттерні арқылы:
- Текст
- HTML
- (PDF кеңейтуге болады)

Есеп:
- Динамикалық бөлімдер қосу
- Стиль беру (фон түсі, шрифт, өлшем)
- Әртүрлі форматта экспорт

---

# 3️⃣ Prototype – Game Character

Character:
- Денсаулық
- Қару
- Броня
- Қабілеттер

Deep Copy жасалады:
- Weapon clone
- Armor clone
- Skill clone

Бұл ойын кейіпкерлерін шаблон арқылы тез жасауға мүмкіндік береді.

---

## Қорытынды

Бұл жоба:
- Потокобезопасный Singleton
- Кеңейтілген Builder
- Deep Clone Prototype

паттерндерін практикалық түрде көрсетеді.
